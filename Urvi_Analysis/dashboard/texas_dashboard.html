What the Texas split is doing

Texas rows were extracted from the WWC state one-hot fields, then the urbanicity context was carried forward from the enriched base.
All Texas rows: 1,812 Urban-tagged rows: 1,250 Rural-tagged rows: 436 Rows tagged both: 387 Missing urbanicity: 494
  • The All Texas tab keeps every Texas-coded finding.
  • The Urban Texas and Rural Texas tabs are derived from WWC urbanicity tags and can overlap when the underlying row is multi-tagged.
  • For mixed rows, the page uses fractional weights so the split does not overcount the same record multiple times inside each lens.

  • Urban-tagged Texas rows: 1,250.
  • Rural-tagged Texas rows: 436.
  • Rows tagged both rural and urban: 387.
