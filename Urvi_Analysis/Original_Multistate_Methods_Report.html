Original WWC State Signal: Thought Process, Math, and Results

Built for ASA Data Challenge | Uses only original WWC export, not teammate-aggregated file.

Thought Process

We needed to answer "what works for whom" without over-claiming causality from aggregated evidence. The core risk was signal blur: many rows list multiple states, so naive state counts and averages can overstate or misattribute signal.

So we built two complementary views:

All state-coded rows
Multi-state-only rows (decomposition view)

This lets us check if patterns are stable or shift when we isolate multi-state evidence.

Exact Mathematical Methods

1) State extraction from original one-hot flags

For each finding row i, define state set:

S_i = { s : s_Region_State_s = 1 }

Let k_i = |S_i|. Keep rows with k_i > 0. Multi-state subset M = { i : k_i ≥ 2 }.

2) Exploded state view and weights

Create exploded pairs (i,s) for each s ∈ S_i. Assign row weight:

w_(i,s) = 1 / k_i

This ensures each original row contributes total weight 1 across all its states.

3) State metrics (scope = all_rows or multistate_only)

findings_n(s) = |{ findingID_i : (i,s) in E_scope }|

studies_n(s) = |{ studyID_i : (i,s) in E_scope }|

interventions_n(s) = |{ interventionID_i : (i,s) in E_scope }|

weighted_finding_contribution(s) = Σ_(i,s in E_scope) w_(i,s)

contribution_share_pct(s) = 100 * weighted_finding_contribution(s) / Σ_s' weighted_finding_contribution(s')

weighted_outcome_sample(s) = Σ_(i,s in E_scope) w_(i,s) * outcome_sample_i

mean_effect_size_wwc(s) = mean(effect_i)

pct_positive_effect_size(s) = 100 * mean(1[effect_i > 0])

mean_improvement_index(s) = mean(improvement_i)

Means and positive-rate are over non-null numeric values.

4) Target-state intervention summaries (NY, NJ, CA, TX)

For target state t in multi-state subset:

R_t = { i in M : t in S_i }

Then group by intervention to compute findings, studies, mean effect, median effect, positive-rate, and mean improvement.

How This Connects to the ASA Task

These outputs are directly aligned to "what works for whom": they show how effectiveness changes across contexts (states), and where evidence density is heavy vs thin. This supports a concrete transportability question:

Do interventions validated in one state context retain effectiveness in another context?
